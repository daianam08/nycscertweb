Teaching Methods

Creating an unplugged activity

This one was fun to make with the people in my group. I am not one for unplugged activities but I like the one we created because it is a good starting point when teaching concepts like Random and conditionals in a fun way.

Task
Students will be tasked to create their own game that includes 1-2 players, 2 dice, and conditionals similar to the example.
The game of Craps will be shown as an example (we will use a different name in class so we avoid teaching gambling).

Differentiation
* Mild: Create a simple game that involves one of either IF or ELSE.
* Medium: Create a game that involves both IF and ELSE.
* Spicy: Create a game that involves at least 2 nested IF or ELSE statements.
